# severus-server
severus server for writing data to blockchain.

### Requirements
* macOS or Ubuntu
* Node.js
* npm

### Usage
**install**
```bash
npm install -g TakutoYoshikai/severus-server
```

**run server**
You have to create config.json from example.config.json before you run server.

```bash
severus-server <CONFIG.JSON>
```
